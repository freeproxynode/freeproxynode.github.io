一、了解小火箭：iOS端科学上网的利器

1.1 什么是小火箭？

小火箭（Shadowrocket）是一款基于规则分流的网络代理工具，专为苹果iOS系统开发。其强大之处在于：

  • 支持多种代理协议（Shadowsocks、VMess、VLESS、Trojan等）；

  • 支持自定义规则与URL重写；

  • 支持HTTP、HTTPS抓包、调试功能；

  • 内建测速、日志查看、连接跟踪等工具。

虽然小火箭本身需要在App Store购买（约30元人民币），但其提供的稳定性和专业性在所有同类工具中堪称顶级。

1.2 小火箭适用于哪些人？

  • 留学生、海外华人：反向访问国内资源

  • 程序员与网络技术人员：需要访问GitHub、StackOverflow等被限制资源

  • 跨境电商用户：用于登录或管理海外站点

  • 国际媒体访问者：需要接触不同国家的新闻信息源

二、VLESS协议简介：未来科学上网的核心协议

2.1 什么是VLESS？

VLESS是V2Ray项目推出的一种新一代网络代理协议，设计初衷是取代原本的VMess协议，解决其在隐私、性能和扩展性方面的瓶颈。

2.2 VLESS的技术优势

无状态连接VLESS采用无状态传输，服务器压力更小，响应速度更快

协议透明更容易和Web服务融合（如伪装成HTTPS、WebSocket等）
兼容性强可配合Trojan、Reality、XTLS等使用

三、小火箭支持VLESS了吗？官方确认支持

许多用户疑惑：“小火箭是否真的支持VLESS协议？”实际上，随着小火箭的持续更新，其已正式原生支持VLESS协议，包括主流的传输层（如TCP、TLS、WS、gRPC等）。只要你使用的是较新版本（一般指2022年之后发布的），就可以在添加节点时选择“VLESS”。

这也意味着，iOS用户无需越狱、无需配置第三方工具，即可使用VLESS协议连接高速稳定的代理节点。

四、小火箭配置VLESS节点的详细步骤

4.1 第一步：下载安装小火箭

  • 打开App Store（需美区账号）

  • 搜索“Shadowrocket”

  • 点击购买并下载安装（目前售价约$2.99）

注意：因政策原因，国区App Store可能无法下载。你可以通过注册一个美区Apple ID进行购买。

4.2 第二步：获取VLESS节点信息

你可以从以下方式获取VLESS节点：

  • 向机场服务商购买节点（常见如Just My Socks、Clash订阅平台等）

  • 自行搭建V2Ray/VLESS服务器（如通过AWS搭建，参考前一篇教程）

  • 向可信的朋友获取节点分享链接

节点信息一般包括：

  • 地址（IP或域名）

  • 端口号

  • UUID（用户唯一识别码）

  • 加密方式（通常为none）

  • 传输协议（TCP、WS、gRPC）

  • TLS设置（是否开启加密）

  • 额外参数（如路径、host、sni等）

4.3 第三步：添加VLESS配置到小火箭

方法一：手动添加

  1. 打开小火箭，点击主界面右上角“+”

  2. 选择“类型”：VLESS

  3. 填写以下信息：

项目示例
地址vless.example.com
端口443
UUID123e4567-e89b-12d3...
加密方式none（常见）
传输协议TCP / WS / gRPC
TLS设置开启（推荐）
伪装域名cloudflare.com（如有）
路径/host等按照提供者填写

  1. 点击“完成”，返回主界面即可看到新添加的节点

方法二：导入配置链接

许多服务商会提供形如 vless:// 的链接：

pgsql
vless://uuid@server:port?encryption=none&security=tls&type=ws&path=/ray#MyVLESS

在小火箭中选择“导入链接”粘贴即可自动生成配置。

4.4 第四步：连接VLESS节点

  1. 返回主界面，勾选你刚添加的VLESS节点

  2. 点击下方“连接”按钮

  3. 状态栏出现VPN图标，表示连接成功

  4. 打开浏览器尝试访问Google或YouTube进行测试

FAQ

Nekoray 的内置延迟测试为什么与系统 ping 不同？
客户端的延迟测试通常基于 TCP 或特定探测包，而系统 ping 使用 ICMP，两者网络路径与优先级可能不同。此外，防火墙或服务器对 ICMP 的响应可能被限制，导致测试结果存在差异。
Hiddify Next 是否兼容 Clash 或 V2Ray 配置文件？
Hiddify Next 支持部分 Clash 与 Xray 订阅格式，但建议使用其官方订阅格式以获得完整兼容。若订阅转换异常，可使用 Hiddify 提供的在线转换工具重新生成配置文件导入。
SSR 混淆方式如何提高抗封锁能力？
SSR 混淆可将流量伪装成正常 HTTPS 或 HTTP 数据，降低被 DPI 检测风险。合理选择协议和混淆参数组合，可以提高节点稳定性和长期可用性，保证科学上网体验顺畅。
Mihomo 是什么？
Mihomo 是 Clash Premium 的开源替代项目，兼容原版 Clash 配置文件并扩展了部分新功能。它性能优异、内存占用低，适用于希望自定义构建代理环境的高级用户。
VLESS + XTLS 模式在高延迟网络中表现如何？
XTLS 模式通过减少握手和加密开销降低延迟，在高延迟环境中保证连接稳定。结合策略组和多出口节点使用，可提供更流畅的网页浏览、视频播放和游戏体验。
VLESS 是否比 Trojan 更稳定？
两者都非常稳定，但 VLESS 在多平台兼容性上略优，而 Trojan 在 HTTPS 伪装和 TLS 安全性上更强。选择取决于服务器环境与具体需求，两者皆为高性能协议。
SSR 是否支持 UDP 协议转发？
原版 SSR 并不完全支持 UDP 转发，但部分魔改版本（如 SSRR）支持该功能。可在配置中开启 udp-over-tcp 模式或使用 Socks5 UDP relay 方案实现游戏等实时通信的代理需求。
Vmess 是什么协议？
Vmess 是 V2Ray 的核心传输协议之一，提供高效的加密通信。它在客户端与服务器之间建立安全通道，支持多种传输方式，如 TCP、WebSocket 和 gRPC 等。由于其加密特性，Vmess 常用于绕过网络限制与防火墙检测。
Hiddify Next 如何设置用户带宽限制？
管理员可为每个用户设置独立带宽和流量配额，超出时可自动降速或阻止访问。这样可确保服务器资源合理分配，避免单用户占用过多带宽，提高整体网络效率和公平性。
VMess 的安全性是否优于 Shadowsocks？
VMess 拥有动态验证机制和更复杂的握手协议，相较于静态密钥的 Shadowsocks 安全性更高。它能有效抵御重放攻击和 DPI 检测，更适合长期使用。
